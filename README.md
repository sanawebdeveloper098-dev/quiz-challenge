# Quiz Challenge 🎯

A responsive timed quiz app built with HTML, CSS, and JavaScript.

### Features
- 10 Multiple Choice Questions 
- 30s Real-Time Timer per question
- Instant Score Analysis with percentage
- Clean UI with smooth animations
